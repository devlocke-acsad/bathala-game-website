# Bathala Website Colors and Fonts

## Fonts Used

1. dungeon-mode
- Type: Custom display font
- Source file: assets/fonts/dungeon-mode.ttf
- Main usage: Headings and display text

2. pixeled
- Type: Custom pixel font
- Source file: assets/fonts/PixeledEnglishFont.otf
- Main usage: Hero/title retro styling

3. Crimson Text
- Type: Serif body font (Google Fonts)
- Main usage: Body copy and long-form text

## Notable Gradients and Effects

1. Gold gradient
- linear-gradient(135deg, gold-dark, gold, gold-light)

2. Dark background gradient
- linear-gradient(180deg, 0 0% 4% to 0 0% 8%)

3. Section gradient
- linear-gradient(180deg, 0 0% 6%, 0 0% 9%, 0 0% 6%)

4. Glow and text-shadow
- Gold glow shadows and gold text-shadow are used for emphasis

## Notes for PowerPoint Consistency

1. Heading font: Use dungeon-mode (or a similar fantasy display font if unavailable in PPT).
2. Body font: Use Crimson Text.
3. Primary visual identity: Dark charcoal background with warm gold highlights.
4. Supporting accents: Emerald, cyan, amber, and red for element/category coding.
5. For quick slide theming:
- Background: #0F0F0F
- Main text: #E2D9CC
- Primary accent: #E8AB30
- Secondary accent: #F6C855
